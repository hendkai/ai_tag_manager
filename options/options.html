üìö Understanding AI Models & Token Limits

What are tokens?
Tokens are pieces of text that AI models process. When analyzing your tags, the AI needs tokens for both reading your tags (input) and writing suggestions (output).

Why does this matter?
Each AI model has a maximum output limit (how much it can write back). If you have many tags (500+), the AI might find hundreds of similar groups, but if it hits its token limit, it can only return a fraction of them.

‚ö†Ô∏è What happens with insufficient tokens?
If you have 1000 tags and use a smaller model, the AI might only return 30-50 groups instead of 100+. The extension will automatically recover as many groups as possible from truncated responses, but you'll still miss many similarities. Use the recommended model for your tag count!

üî• For Deep Analysis Mode:
Deep Analysis sends ALL tags in ONE request, which requires MUCH more output capacity. With 1000+ tags, only GPT-4o's 16K output limit can handle the full response. Smaller models will be truncated and you'll need to run the analysis 2-3 times to find all similarities.
